EOS API Connect has been upgraded!

Changes have been made to our API increasing security, simplifying processes and improving request efficiency.
  • From now on no more API key in the endpoint URL. We have added a layer of security and the API key should be provided in the header of the request.
  • Our field management option now allows easier and smoother images and updated satellite information retrieval. It is no longer needed to bother yourself parsing the field information and coordinates in every single request. We keep that information for you and simply using a field identifier, you would be able to retrieve the information you need.
  • The GeoTIFF format is now supported in the output including the georeferencing information as metadata in the images of the vegetation indexes.

Other popular features allow you to:

 
 

Analyze data with NDVI and 17 more indices.
 
 
 

Download satellite imagery from 9 different sources.
 
 
 

Download bands of any scene and images of any area.
 
 
 

Get soil moisture at 250m resolution since 2015 or 2002*
 
 
 

Get a 7-day or 14-day weather forecast depending on a source.
 
 
 

Get advanced high accuracy weather forecast and historical weather data**
 
 
 

Divide your field into more and less productive areas for VRA with vegetation maps.
 
 
 

Learn the quality of your field based on NDVI historical trends with productivity maps.
